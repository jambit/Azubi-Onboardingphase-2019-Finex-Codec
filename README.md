## How to run the code ?
####1. Build project
Open any type of terminal and enter following command
#####Windows (cmd)

```console
C:\project\directory> gradlew.bat build
```
#####Linux  
```console
foo@bar:~$ ./gradlew build
```
####2. Run project
#####Windows (cmd)
```console
C:\project\directory> gradlew.bat run --console=plain
```
#####Linux
```console
foo@bar:~$ ./gradlew run --console=plain
```
